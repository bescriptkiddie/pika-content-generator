# M3.5: 风控网关

## 职责

量化场景专用。在 M3 生成交易信号后、M4 执行下单前，进行风控校验。

**不可被绕过。**

## 风控规则

### 通用规则
- 单笔最大仓位比例
- 单日最大亏损比例
- 最大持仓数量

### A股专用
- 涨跌停限制（不追涨停、不卖跌停）
- 禁止交易时段（集合竞价等）
- 持仓集中度上限

### 加密货币专用
- 最小流动性要求
- 最大滑点容忍度
- 极端波动熔断（24h波动 > X% 暂停）
- 资金费率异常告警

## 处理流程

```
M3 交易信号
    ↓
风控校验 ──→ 通过 → M4 执行
    │
    ├── 降级 → 调整仓位后执行
    ├── 拒绝 → 记录原因，不执行
    └── 告警 → 通知人工确认
```

## 关键设计

- 所有拒绝/降级操作必须记录日志
- 风控规则可动态配置，但修改需要人工确认
- 默认 require_confirmation: true（需人工确认后才下单）
